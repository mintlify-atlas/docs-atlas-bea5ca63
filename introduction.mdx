<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your data warehouse running in minutes with Docker
  </p>

  <Steps>
    <Step title="Clone the repository">
      Download the project and navigate to the directory:
      
      ```bash
      git clone https://github.com/LeoBellier/sql-data-warehouse-project.git
      cd sql-data-warehouse-project
      ```
    </Step>

    <Step title="Set up environment variables">
      Create a `.env` file with your PostgreSQL credentials:
      
      ```bash
      POSTGRES_USER=warehouse_admin
      POSTGRES_PASSWORD=your_secure_password
      POSTGRES_DB=datawarehouse
      ```
    </Step>

    <Step title="Start the PostgreSQL container">
      Launch the database with Docker Compose:
      
      ```bash
      docker-compose up -d
      ```
      
      <Note>The container mounts the datasets directory at `/datasets` for CSV imports.</Note>
    </Step>

    <Step title="Initialize the schemas">
      Connect to the database and create the Bronze, Silver, and Gold schemas:
      
      ```bash
      psql -h localhost -U warehouse_admin -d datawarehouse -f scripts/SchemaCreator.sql
      ```
    </Step>

    <Step title="Load the data">
      Run the ETL procedures to populate your data warehouse:
      
      ```sql
      CALL bronze.load_bronze();
      CALL silver.load_silver();
      ```
      
      <Accordion title="Expected output">
        ```
        NOTICE: [14:32:15] load bronze started
        NOTICE: Step [load bronze.crm_cust_info] completed successfully (1 s)
        NOTICE: Step [load bronze.crm_prd_info] completed successfully (0 s)
        NOTICE: Step [load bronze.crm_sales_details] completed successfully (2 s)
        NOTICE: load bronze finished in [3] seconds
        ```
      </Accordion>
    </Step>

    <Step title="Query the gold layer">
      Run analytics queries on the star schema:
      
      ```sql
      SELECT 
        p.category,
        COUNT(DISTINCT f.customer_key) as customer_count,
        SUM(f.sales_amount) as total_sales
      FROM gold.fact_sales f
      JOIN gold.dim_product p ON f.product_key = p.product_key
      GROUP BY p.category
      ORDER BY total_sales DESC;
      ```
    </Step>
  </Steps>
</div>
